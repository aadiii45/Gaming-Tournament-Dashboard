# Gaming Tournament Dashboard

## Project Overview
The Gaming Tournament Dashboard is a database-driven web project designed to manage gaming tournament information. It allows users to view tournament details and manage player and match data using a structured database system.

## Technologies Used
- HTML
- CSS
- SQL
- MySQL (Database)

## Features
- Tournament information display
- Player data management
- Database integration using SQL queries
- Simple and user-friendly interface

## Project Structure
Gaming-Tournament-Dashboard
│
├── index.html
├── style.css
├── database.sql
└── other project files

## How to Run the Project
1. Install XAMPP or any local server.
2. Import the SQL database into MySQL.
3. Place the project folder in the server directory.
4. Run the project on localhost.
